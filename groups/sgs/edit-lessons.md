# SGS Edit Lessons (Andy's learning loop)

Append-only log. Andy reads this before every batch and writes one entry after every
shipped or triaged ticket (via `ship-sgs.ts log`). Patterns that repeat → promote them
into `CLAUDE.md` as standing rules so they compound into skill.

Format per entry:
- **What:** what was requested / done
- **Lesson:** what worked, what the customer preferred, what to do differently
